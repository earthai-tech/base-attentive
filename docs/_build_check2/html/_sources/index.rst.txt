BaseAttentive Documentation
===========================

BaseAttentive is a configurable kernel for probabilistic forecasting with
attentive sequence-to-sequence models. The project focuses on flexible model
composition, backend-aware runtime selection, and practical forecasting
workflows for static, dynamic, and known-future features.

What you will find here
-----------------------

The documentation is organized around the public user workflow:

- installation and backend selection
- architecture and data-shape expectations
- usage patterns for the main ``BaseAttentive`` model
- API reference for runtime helpers, validation utilities, and learner mixins
- development and documentation build guidance

Public API at a glance
----------------------

The main public entry points are:

.. code-block:: python

   from base_attentive import BaseAttentive
   from base_attentive import get_backend, set_backend
   from base_attentive import (
       get_available_backends,
       get_backend_capabilities,
   )

``BaseAttentive`` is the primary model class. The backend helpers let you
inspect and control the active Keras runtime, while the validation helpers
in ``base_attentive.validation`` provide lightweight tensor-shape utilities.
